Apps

What you get

Every app is a native desktop window, launched from the nOffice shell. Switch between them instantly, share content between apps.

nWrite — Word Processor

Write documents with tables, images, footnotes, and comments. Track changes, export to PDF or DOCX. Clean, distraction-free editing.

nSheet — Spreadsheet

Work with large datasets, build charts, run formulas. Supports millions of rows. Import and export CSV, XLSX, ODS.

nSlides — Presentations

Create slide decks with transitions and animations. Export to PDF and PPTX. Present directly from the app.

nImg — Image Editor

Paint, edit, and composite images. Layers, brushes, shapes, filters. Open and save all common formats.

nCode — Code Editor

Full IDE with Monaco Editor. Syntax highlighting, Git integration, multi-language support. Built-in language servers for Python, C++, Rust, Go, Ruby.

Built-in Assistant

Every app includes an assistant that can answer questions, rewrite text, generate content, explain code, summarize documents, and translate between languages. Runs locally — no internet needed.
